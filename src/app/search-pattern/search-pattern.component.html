About this Tool

The Qmine tool is a cutting edge tool that allows users to upload and process trace logs. It analyzes trace files by searching through them for Regex patterns called Qualitative Regular Expression Template (QRET). By analyzing these tools files, one can detect anomalies or any point of interest that the user wishes to find from the system. It separates the trace into separate finite state machines (FSM) that work on independent parts of the data. The trace separation is done using a sliding window method, where each window will take a set subset of the trace and try to search for the QRET pattern there. Afterwards, the system will take any matches from the search and will be displayed on this tool.
